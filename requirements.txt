# Pinned runtime dependencies for the bvidfe library + CLI + Streamlit
# Cloud deployment.
#
# Streamlit Community Cloud reads this file at deploy time. The leading
# "." entry triggers `pip install .` so Streamlit Cloud installs the
# bvidfe package itself from pyproject.toml — without it, app.py would
# import-fail because the cloud worker's PYTHONPATH does not include
# the repo's src/ tree by default.
#
# Headless library users (no Streamlit, no GUI) can ignore the streamlit
# line — it is pure-Python and adds no native build cost. PyQt6 and
# pyvista (desktop GUI) and pytest / pytest-qt / ruff / black (dev
# tools) are intentionally NOT here; they live in pyproject.toml's
# optional-extras and ship via:
#
#     pip install -e ".[gui]"        # GUI only
#     pip install -e ".[dev]"        # dev tools
#     pip install -e ".[all]"        # everything
#
# Versions are late-2024 stable releases inside pyproject.toml's bounds.
# Re-pin from `pip freeze` in the validated build environment if a
# different combination ships.

# Editable install of the bvidfe package itself — REQUIRED by app.py.
.
